# bitnet-int8-rs

Pure-Rust inference for **BitNet b1.58 2B4T** (Microsoft's native 1.58-bit LLM,
W1.58A8), plus a verified numpy reference and a study of **where fully-integer
inference holds and where it breaks**.

```bash
cargo run --release -- "The capital of France is" 20
# The capital of France is Paris. Paris is a city ...
```

## What's here

- **`src/`** — clean Rust implementation. Ternary {-1,0,+1} weights, per-token
  int8 absmax activation quant, integer-exact matmul, GQA, RoPE, squared-ReLU
  FFN, subln, tied LM head, KV cache.
- **`reference/`** — numpy oracle + a fully-integer path (integer RMSNorm, RoPE,
  softmax, KV-cache) with precision knobs. Used to reverse-engineer the I2_S
  weight format and to run the characterization.
- **`reference/FINDINGS.md`** — the write-up.

## Headline results

Measured on the real `ggml-model-i2_s.gguf` weights against a faithful float oracle:

| question | answer |
|---|---|
| Does a fully-integer forward (int8 act + int8 KV + integer attention) track float? | **Yes — token-exact**, final-layer cosine 0.9997 |
| How cheap can integer softmax be? | A **16-entry** exp LUT still generates identical text |
| Where's the KV-cache cliff? | It's set by quantizer **granularity + zero-point**, not bit-width. On WikiText-2 (10k tok, ref 27.4): int8 KV is free; 2-bit per-vector **and** per-channel-symmetric both collapse (+264–266); **only per-channel *asymmetric* rescues 2-bit (+10)** |
| Mean/median vs absmax activation scaling? | absmax is **load-bearing**; mean/median raise perplexity by **3–6 orders of magnitude** — activations are heavy-tailed (absmax/absmean p90 ≈ 106) and the outliers carry the signal |
| Does it generalize across architectures? | **Yes** — the same surface holds on Falcon3-1B-1.58 (int8 free, only per-channel-asym survives 2-bit, robust scaling catastrophic; Key per-channel bias 0.79σ in both models) |

The contribution in one sentence: *a native-1-bit model runs a fully-integer
forward token-exactly, and its integer inference is limited not by low-bit
attention/softmax/KV but by the fact that its int8 activations are
outlier-carried — so outlier-robust scaling, not bit-width, is what breaks it.*

See [`reference/FINDINGS.md`](reference/FINDINGS.md) for the full tables (incl. the
held-out corpus perplexity and per-channel KV study), method, and honest
limitations, and [`CLAUDE.md`](CLAUDE.md) for the verified model facts and the
I2_S on-disk format.

## Reproducing the paper

The per-token NLL arrays that back every table and figure are committed under
`reference/_ppl*`, so the figures regenerate **without** the model:

```bash
cd reference && python3 make_figures.py   # -> reference/figures/*.pdf
```

To re-run the experiments from scratch (needs the weights above):

```bash
cd reference
pip install numpy tokenizers
python3 prepare_weights.py     # one-time: unpack I2_S -> int8 memmap (~2 GB, gitignored)
python3 run_ppl2.py            # KV / activation sweep on WikiText-2
python3 analyze_ci.py          # bootstrap CIs -> the numbers in Tables 1, 2, 6
```

The held-out WikiText-2 slice is regenerated by `reference/wikitext_loader.py`
(WikiText-2 is CC BY-SA; we point to the source rather than redistribute it).
